:: WELCOME TO THE ARCHIVE

hi. welcome. the internet forgets a lot of what people make — sites go down, links rot, repos go private. Ledger is a small corner where the stuff you build gets to stick around.

ship a project here and it gets a name, a date, and a line in the catalogue so someone — maybe years from now — can still find it and see what people were making in 2026. it's a hack club ysws, it's free, and it lives for decades on purpose. new here? you're in the right place.

have questions or just want to talk? come hang out in #ledger on the Hack Club Slack — we're friendly and we don't bite.

:: SHIP IT TO THE ARCHIVE

built something? file it here. once it's accepted your project gets a permanent entry in the catalogue — name, description, date filed, and a link the future can follow. takes about 4 minutes. first ship? even better.

» SHIP NOW «
[!] before you ship: your project has to actually run and have a real name + a short description. no spam, no forks where you changed nothing. if you built it yourself then ship it, doesn't matter if it's small or bad. we want it.

:: FREQUENTLY ASKED

how long does an entry stay? indefinitely. the whole point is entries don't expire when the hype does.

is this free? yes, this is all 100% free — funded by a nonprofit called Hack Club.

what's a ship? a ship is basically making a project and submitting it. to ensure the ship is worthy of possessing a place in such archive, make sure your project has a non-generic title, a 2–3 sentence description, a readme, and all the files in a github repo. also a demo link is needed — it should be a link to a website for example, or if that's not possible, a github release.

what is a review? we just check if your project meets the guidelines, and it usually takes less than 2 days.
